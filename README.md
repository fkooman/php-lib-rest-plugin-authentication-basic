# Introduction
This is a plugin for supporting HTTP Basic Authentication in your REST 
service based on `fkooman/rest`.

# Installation
You need to require `fkooman/rest-plugin-basic` in your `composer.json` and run
`composer update`.

# Usage
See `example` directory.
